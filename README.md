# Python热门游戏推荐系统

## 核心功能
- 用户登录注册
- 游戏信息展示
- 社区互动平台
- 游戏资讯浏览
- 意见反馈功能
- 游戏数据可视化大屏
- 游戏信息管理
- 评分预测模型
- 社区互动管理
- 意见反馈管理
- 用户管理
- 系统管理

## 适用场景
- 游戏爱好者获取最新游戏资讯
- 游戏开发者分析游戏数据
- 社区管理员管理社区互动
- 游戏公司进行市场分析

## 技术特点
- 基于Django框架，实现前后端分离
- 使用MySQL数据库存储数据
- 利用Vue.js进行前端开发
- 支持用户管理、游戏信息管理、社区互动管理等功能
- 提供数据可视化大屏展示游戏数据
- 包含评分预测模型，为用户提供个性化推荐
